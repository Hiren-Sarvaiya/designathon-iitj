# LOGIC-LAB Frontend

A gamified logical reasoning platform build with React, Tailwind CSS, and Framer Motion for the Designathon.

## Getting Started

1. **Install Dependencies** (if not already done)
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to `http://localhost:5173`.

## Features Implemented
- **Landing Page**: Immersive parallax design with neon aesthetics.
- **Learning Hub**: Dashboard with progress tracking and unlocked modules.
- **Predictor Hub**: Pattern recognition module with filtering and level selection.
- **Puzzle Interface**: Fully interactive logic puzzles with validation, hints, and confetti rewards.
- **Leaderboard**: Global ranking system with user highlighting.
- **Resources**: Downloadable content area for offline learning.

## Tech Stack
- **Core**: React 19, Vite
- **Styling**: Tailwind CSS 3.4, clsx, tailwind-merge
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Extras**: React Confetti
